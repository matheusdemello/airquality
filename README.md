# Ak-shumkar Nest — GitHub Pages

This is a static site (HTML/CSS/JS) ready for GitHub Pages.

## Upload to GitHub Pages

1. Create a new GitHub repository (public is simplest for Pages).
2. Upload all files in this folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/` (images)
3. In GitHub: **Settings → Pages**
   - **Source**: Deploy from a branch
   - **Branch**: `main` / `(root)`
4. Your site will be available at:
   `https://<username>.github.io/<repo-name>/`

## Notes

- The checklist uses `localStorage` to save progress on the same device/browser.
- All text and images come from the provided report PDF.
